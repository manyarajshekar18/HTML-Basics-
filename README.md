# HTML Basics

This repository covers the basics of HTML (HyperText Markup Language) used to create and structure web pages. It includes headings, paragraphs, links, images, lists, tables, forms, and semantic elements. Designed for beginners, it helps build a strong foundation in web development with simple examples and practice files.

## Topics Covered
- Introduction to HTML
- HTML Document Structure
- Headings and Paragraphs
- Text Formatting
- Links and Images
- Lists
- Tables
- Forms
- Semantic HTML Tags

## Technologies Used
- HTML5

## Purpose
The main purpose of this repository is to practice and understand the fundamentals of web page creation using HTML.

## How to Use
1. Clone the repository
2. Open the HTML files in any web browser
3. Explore and practice the examples
